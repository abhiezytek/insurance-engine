# PrecisionPro Module and Admin Review Guide

## Required top-level modules
- Benefit Illustration
- YPYG
- Audit
- Admin

## Audit module target structure
- Payout
  - Dashboard
  - Single Policy
  - Bulk Upload
- Addition/Bonus
  - Dashboard
  - Single Policy
  - Bulk Upload

## Dashboard fix
- Module cards/tiles must be clickable and route to working pages.

## Admin capabilities
- create new user
- activate/deactivate user
- manage subscriptions
- grant/revoke module access
- grant/revoke product access
- configure new products
- manage formula/rule versions
- manage lookup tables
- manage core system mappings
- manage templates and downloads
- preserve audit trail
